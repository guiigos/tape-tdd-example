## Usage
### Dependencies
Install dependencies and modules.

```bash
$ npm install
```

### Scripts
After installing the dependencies the application is ready to run.

```bash
# Test
$ npm run test
$ npm run test:nyan

# Coverage
$ npm run coverage
```
